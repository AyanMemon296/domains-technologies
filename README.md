A premium, minimalist tech encyclopedia and showcase platform designed for developers. **Domains Technologies** serves as a sleek, single-page visual guide to modern programming languages, development frameworks, and cutting-edge tools.

👉 **Live Website:** [https://AyanMemon296.github.io/domains-technologies/](https://AyanMemon296.github.io/domains-technologies/)

---

## 🌓 Dark & Light Mode

The platform features a strictly monochrome, "Luxury Tech" aesthetic. The transition between themes is instantaneous and smooth, saving your preference locally for a consistent experience.

- **Light Mode**: Primary background `#FFFFFF` | Text & Icons `#000000`
- **Dark Mode**: Primary background `#000000` | Text & Icons `#FFFFFF`

## 🚀 Key Features

- **📂 Category Tabs**: Seamlessly switch between 6 specialized categories:
  - Languages
  - Web Development
  - Mobile App Development
  - Data Science
  - AI / ML
  - Tools
- **🔍 Real-time Search**: Instant filtering of technology cards within the active category.
- **✨ GSAP Animations**: High-end staggered entrance and fade transitions for a premium feel.
- **📱 Fully Responsive**: Optimized for Desktop, Tablet, and Mobile devices with an auto-adjusting grid.
- **🛡️ Error Handling**: Intelligent icon fallback system — if a brand logo fails to load, a native developer icon takes its place seamlessly.

## 🎨 Design Philosophy

Built with a **Minimalist First** approach:

- **Typography**: Utilizing _Inter_ for crisp, modern readability.
- **Monochrome Icons**: Brand logos from _Simple Icons_, _Devicon_, and _Lucide_ are filtered dynamically to adhere to the B&W theme.
- **Space & Border**: Generous whitespace and subtle 1px borders create a sophisticated, uncluttered interface.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5 🏗️
- **Styling**: Vanilla CSS3 with Custom Properties (Variables) 🎨
- **Logic**: Vanilla JavaScript (ES6+) ⚡
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform) 🪄
- **Icons**:
  - [Lucide Icons](https://lucide.dev/) for UI elements
  - [Devicon](https://devicon.dev/) for tech brands
  - [Simple Icons](https://simpleicons.org/) for modern brand accurate logos
